Прототип - паттерн, порождающий объекты.

Задает виды создаваемых объектов с помощью экземпляра-прототипа
и создает новые объекты путем копирования этого прототипа.

Использовать шаблон прототип можно когда:
1. Классы, экземпляры которых нужно создать,
определяются во время выполнения прграммы.

2. Для избежания построения иерархий классов,
фабрик  или параллельных иерархий классов.

3. Экземпляры класса могут находится в одном
из немногих возможных состояний.
